# 电商老板热点挖掘机

## 快速开始

```bash
# 安装依赖
pip install requests beautifulsoup4

# 获取今日热点
python scripts/ecommerce_hotspot_miner.py

# TOP10热点
python scripts/ecommerce_hotspot_miner.py --limit 10

# JSON格式输出
python scripts/ecommerce_hotspot_miner.py --format json
```

## 数据源

| 来源 | URL | 特点 |
|:---|:---|:---|
| 虎嗅 | huxiu.com | 财经、产业、商业分析 |
| 36氪 | 36kr.com | 科技、创业、商业 |

## 痛点映射

将通用热点转换为老板能共鸣的内容：

| 痛点 | 触发词 | 权重 |
|:---|:---|:---:|
| 利润薄 | 赚钱、盈利、亏损、订单 | +25 |
| 流量贵 | 流量、获客、广告、推广 | +25 |
| 成本高 | 成本、涨价、房租、人工 | +25 |
| 平台压榨 | 平台、抽成、封号、规则 | +22 |
| 资金压力 | 融资、贷款、账期、违约 | +22 |
| AI焦虑 | AI、智能、替代、裁员 | +22 |
| 转化难 | 转化、销售、复购 | +15 |
| 消费降级 | 消费、降级、省钱 | +15 |

**电商加权**：标题含"电商/跨境/老板"等词 +15分

## 热点 → 选题对照

| 热点 | 关联痛点 | 选题方向 |
|:---|:---|:---|
| 黄金暴跌 | 资金压力/消费降级 | "黄金都跌了，电商老板怎么保值？" |
| 酒店亏本 | 成本高 | "酒店都撑不住了，你的店呢？" |
| AI上位 | AI焦虑 | "90后用AI替代老人，老板慌不慌？" |
| 北京6块钱吃饭 | 消费降级 | "消费者都这样省了，你还敢提价？" |

## 命令参数

| 参数 | 说明 |
|:---|:---|
| `-l, --limit` | 返回数量 (默认20) |
| `-f, --format` | 输出格式 (text/json) |
| `-o, --output` | 输出到文件 |
| `-a, --analyze` | 分析单个关键词 |
| `-q, --quiet` | 简洁模式 |

## 注意事项

- 百度热搜/微博热搜需要登录，无法直接抓取
- 建议虎嗅+36氪组合，覆盖最全面
- 如遇网络问题，检查网络连接
